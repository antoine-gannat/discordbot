# discordbot
A discord bot in node-js, full of memes and stuff

#	Installation	#

1.	Install node-js at https://nodejs.org/en/

2.	Type in a terminal:

```sh
git clone https://www.github.com/antoine-gannat/discordbot
cd discordbot
npm i
```

#	Execution	#

Type in a terminal:

```sh
node index.js
```
